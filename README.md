# Movie Recommendation System using Spherical K-Means

## Overview

This project implements a movie recommendation system using the MovieLens dataset. Movies are clustered using the Spherical K-Means algorithm based on similarity features, allowing users to discover related movies through an interactive interface.

## Features

* Movie clustering using Spherical K-Means
* Recommendation generation based on cluster similarity
* Interactive user interface
* MovieLens dataset integration
* Data preprocessing and feature engineering

## Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Scikit-Learn
* Spherical K-Means
* MovieLens Dataset

## Workflow

1. Data Collection from MovieLens Dataset
2. Data Cleaning and Preprocessing
3. Feature Extraction
4. Spherical K-Means Clustering
5. Recommendation Generation
6. User Interface Development

## Results

Movies with similar characteristics are grouped into clusters, enabling more relevant recommendations and improved content discovery.
